# localhost-viewport-tester

## Steps to Use:

1. Clone the repo
2. Run the `npm i` on the base directory
3. Run `npm run`

### THATS IT !!!!

## Usage:
On the sticky header, enter the URL, height and width that you want to see your site in. 
Then click "Open".

## Defaults: 
URL: http://localhost:3001 <br>
Height: 600px <br>
Width: 320px
